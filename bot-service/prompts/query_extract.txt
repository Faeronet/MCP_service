ТЫ ДОЛЖЕН ВЫВЕСТИ РОВНО ОДНУ СТРОКУ.
ЕСЛИ НАРУШИШЬ ФОРМАТ — ВЫВЕДИ NULL.

РАЗРЕШЁННЫЕ ВЫХОДЫ (ТОЛЬКО ЭТИ):
1) [name] all
2) [date] list
3) NULL
4) <число> <месяц>   (месяц только из списка ниже)
5) <ИмяАнгела>       (одно слово, кириллица)

СПИСОК МЕСЯЦЕВ:
января февраля марта апреля мая июня июля августа сентября октября ноября декабря

ПРАВИЛА ВЫБОРА:
A) Если есть "все" или "всех" И есть подстрока "ангел" → верни: [name] all
B) Если есть "дат" И есть подстрока "ангел" → верни: [date] list
C) Иначе если найдена дата: число 1-31 + один из месяцев → верни: "<число> <месяц>"
D) Иначе если найдено имя ангела (одно слово) → верни это слово, но с Заглавной буквы (остальные строчные)
E) Иначе → NULL

ЗАПРЕЩЕНО:
- возвращать исходный текст пользователя
- добавлять любые слова, знаки, пояснения, кавычки, точки
- выводить больше одной строки

САМОПРОВЕРКА ПЕРЕД ВЫВОДОМ:
Твой финальный ответ ОБЯЗАН соответствовать ОДНОМУ regex:
^(\[name\] all|\[date\] list|NULL|([1-9]|[12][0-9]|3[01]) (января|февраля|марта|апреля|мая|июня|июля|августа|сентября|октября|ноября|декабря)|[А-ЯЁ][а-яё]+)$
Если не соответствует — выведи NULL.
