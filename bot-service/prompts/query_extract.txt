ЗАДАЧА

Из вопроса пользователя извлеки КЛЮЧ ПОИСКА.

Ключ поиска — это главная смысловая сущность, нужная для поиска по текстам.

Ответ должен содержать только ключ поиска, без объяснений.


ОБЩИЕ ПРАВИЛА

1. Ответ должен содержать только ключ.
2. Ответ должен быть коротким.
3. Не добавляй пояснений.
4. Не пересказывай вопрос.
5. Не добавляй слова "ангел", "кто", "какой", "кому", "где", "может", "поможет", "способен", "отвечает", "подвластны", если они не входят в сам ключ.
6. Игнорируй служебные слова вопроса.
7. Извлекай только главную сущность для поиска.


ОПРЕДЕЛЕНИЕ ТИПА ЗАПРОСА

Сначала определи, к какому типу относится вопрос.

ТИП 1 — ЧАСТЬ ТЕЛА ИЛИ ОРГАН
Если вопрос о лечении, помощи, исцелении, восстановлении части тела, конечности или органа,
то ключом является ЧАСТЬ ТЕЛА или ОРГАН.

Примеры:
какой ангел может исцелить ноги -> ноги
кто исцелит ноги -> ноги
кто поможет с ногами -> ноги
кто исцелит руки -> руки
кто поможет с сердцем -> сердце


ТИП 2 — ВЛАСТЬ, ПОДЧИНЕНИЕ, ВОЗДЕЙСТВИЕ НА КОГО-ТО ИЛИ ЧТО-ТО
Если вопрос о том, кому кто подвластен, кто кому подчиняется, кто над кем имеет власть,
то ключом является вся смысловая конструкция: действие + объект.

Примеры:
кому подвластны цари -> подвластны цари
какому ангелу подвластны цари -> подвластны цари
кому подвластны правители -> подвластны правители


ТИП 3 — ДЕЙСТВИЕ НАД СУЩНОСТЬЮ, ПРОЦЕССОМ ИЛИ ЯВЛЕНИЕМ
Если вопрос о том, кто может убрать, растворить, снять, очистить, разрушить, открыть, закрыть что-то,
то ключом является действие вместе с объектом действия.

Примеры:
какой ангел может растворить все кармические долги -> растворить кармические долги
кто поможет снять порчу -> снять порчу
кто может убрать страх -> убрать страх
кто способен очистить карму -> очистить карму


ТИП 4 — ПОИСК ИМЕНИ ПО ШАБЛОНУ
Если вопрос имеет вид:
где обитает ангел [имя]
где находится ангел [имя]
что делает ангел [имя]
кто такой ангел [имя]

то ключом является только ИМЯ.

Примеры:
где обитает ангел Михаил -> Михаил
где находится ангел Рафаил -> Рафаил
что делает ангел Уриил -> Уриил


ТИП 5 — КАЧЕСТВО, СПОСОБНОСТЬ, СВОЙСТВО, СОСТОЯНИЕ
Если вопрос о качестве, способности, состоянии ума, духовном свойстве,
то ключом является само качество или свойство.

Примеры:
какой ангел влияет на точный ум -> точный ум
какой ангел отвечает за тонкий ум -> тонкий ум
кто помогает развить прозорливость -> прозорливость
кто связан с ясностью мышления -> ясность мышления


ПРАВИЛА ВЫБОРА КЛЮЧА

1. Если вопрос о теле или органах — верни только часть тела или орган.
2. Если вопрос о подвластности, власти или подчинении — верни действие и объект.
3. Если вопрос о снятии, растворении, очищении, разрушении, устранении чего-либо — верни действие и объект.
4. Если после слова "ангел" указано имя собственного существа — верни только имя.
5. Если вопрос о качестве или способности — верни только это качество или способность.
6. Не выбирай субъект вопроса.
7. Не выбирай служебные глаголы.
8. Не выбирай слово "ангел", если только после него не нужно извлечь имя.
9. Сохраняй смысл максимально близко к вопросу.
10. Не заменяй слова синонимами без необходимости.


ФОРМАТ ОТВЕТА

Верни только ключ поиска.
Без кавычек.
Без точки.
Без пояснений.
