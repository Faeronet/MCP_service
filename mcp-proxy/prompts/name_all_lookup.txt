ЗАДАЧА

Проанализируй вопрос пользователя и определи, о каких ангелах из предоставленного списка идёт речь.

Тебе будут даны:

1. Список имён ангелов, расположенных по порядку
2. Вопрос пользователя

Ты должен вернуть только числа — номера ангелов в списке.

Если удалось определить одного или нескольких ангелов, верни их номера.

Если определить ничего нельзя, если ни одно имя не найдено, если ни один номер или позиция не существует, верни:

0

- одно число
или
- несколько чисел через один пробел

3
15
69
72 64 9 2
0

Нельзя выводить:
- слова
- комментарии
- пояснения
- запятые
- точки
- скобки
- метки
- дополнительные строки

Если найдено несколько ангелов, выводи только их номера через один пробел в одной строке.

Если не найден ни один корректный ангел, выводи только:

0

Если нет — верни 0.

Нельзя возвращать имена.

ПРАВИЛО ИМЕНИ

3. вернуть все найденные номера

Если имя в вопросе дано не в именительном падеже, нужно привести его к форме имени из списка.

Например:

- Левиаху → Левиах

- Ариеля → Ариель

- Элемиаха → Элемиах

Если в вопросе нет ни одного найденного имени, и никакой другой корректной позиции тоже нет, вернуть:

0

ПРАВИЛО НОМЕРА

Если пользователь прямо указывает номер ангела, нужно вернуть этот номер, но только если такой номер существует в пределах списка.

Примеры:
- расскажи про ангела 3 → 3
- расскажи про ангела под номером 8 → 8
- кто такой ангел №5 → 5
- покажи ангела под цифрой три → 3

Если пользователь указывает несколько номеров, нужно вернуть все существующие номера.

Примеры:
- расскажи про ангелов 2 4 7 → 2 4 7
- покажи ангелов №3 и №9 → 3 9

Если указанный номер не существует в списке, он игнорируется.

Если ни один указанный номер не существует, вернуть:

0

ПРАВИЛО ПОРЯДКОВОЙ ФОРМЫ

0

ПРАВИЛО ПОЗИЦИИ С КОНЦА

Если пользователь спрашивает:
- последнего
- предпоследнего
- третьего с конца
- четвёртого с конца
- пятого с конца

то нужно считать позицию от конца списка и вернуть обычный номер этого ангела в полном списке.

Формула:

если список содержит N имён, то:
- последний = N
- предпоследний = N - 1
- третий с конца = N - 2
- k-ый с конца = N - k + 1

Примеры:
- если в списке 71 имя, то третий с конца → 69
- если в списке 71 имя, то последний → 71
- если в списке 71 имя, то предпоследний → 70

Если пользователь спрашивает несколько позиций с конца, нужно вернуть их все.

Примеры:
- первый с конца и третий с конца
- последний, предпоследний и четвёртый с конца

Если какая-то позиция с конца не существует, она игнорируется.

Если ни одна не существует, вернуть:

0

ПРАВИЛО СЕРЕДИНЫ

- центральный ангел

нужно определить середину списка.

Правило:
- если длина списка нечётная, середина одна → вернуть её номер
- если длина списка чётная, середина неоднозначна, потому что центральных позиции две → этот запрос не даёт результата

Если вопрос просит несколько позиций и среди них есть середина:
- добавляй середину только если она однозначна
- если она неоднозначна, просто не добавляй её

Если кроме неоднозначной середины ничего больше не найдено, вернуть:

0

Примеры:
- при 71 имени середина = 36
- при 70 именах середина не добавляется
